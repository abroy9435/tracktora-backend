# TrackTora Backend

This repository contains the Go/Fiber backend for the TrackTora application tracking system. It is designed to be deployed via Docker on Hugging Face Spaces.

## Features
* **High Performance:** Built with the Fiber v2 framework.
* **Database Integration:** Uses GORM for PostgreSQL management.
* **Authentication:** Secured via JWT.
* **External Integrations:** Adzuna Job Search API.

## Technical Details
* **Language:** Go 1.22
* **SDK:** Docker
* **Deployment:** Hugging Face Spaces
